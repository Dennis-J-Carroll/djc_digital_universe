Core Philosophy

The Essence of Chroma Echo

A Living Digital Canvas

Chroma Echo is a minimalist, immersive digital canvas where every touch generates living color formations that respond to user interaction, audio input, and customizable physics systems. It transforms the screen from a passive display to a living, responsive material—digital clay that remembers and evolves.

Bridging Nostalgia and Innovation

The experience blends the nostalgic appeal of Windows Media Player visualizations with modern generative art and tactile interaction design. It democratizes generative art creation while providing enough depth for professional artists, offering a bridge between performance and preservation.

Universal Accessibility

Simple enough for a child to enjoy, deep enough for an artist to explore for years, and practical enough for a musician to use in tomorrow's concert. The recording system turns ephemeral interactions into permanent, shareable artifacts.

Interaction Pillar 1

Touch Propagation: The Ripple Effect

Single Tap: Color Seeds

Each single tap creates a "color seed" that blooms with perfect radial symmetry, generating expanding circular waves of light that propagate outward from the touch point.

Sustained Pressure: Fractals

Holding pressure generates increasingly complex fractal patterns that evolve and mutate over time, creating organic, branching structures with self-similar characteristics.

Multi-Touch: Interference

Multiple simultaneous touches create complex interference patterns where color waves collide and merge, producing moiré patterns and constructive/destructive wave interactions.

Velocity Sensitivity

Swift gestures create sharp, energetic waves with high frequency and amplitude, while slow movements produce gentle, undulating fields with smooth, flowing characteristics.

Technical Implementation: Each touch point generates a 2D wave equation with customizable frequency, amplitude, and decay parameters. Multi-touch scenarios use superposition principles to create complex interference patterns.

Advanced Integration

Audio Responsiveness: Visualizing Sound

Bass Frequencies: Wave Amplitude

Low frequencies (20-250Hz) directly control wave amplitude and saturation. Deep bass creates large, powerful waves with rich color intensity, while sub-bass generates subtle background undulations.

Treble Frequencies: Pattern Intricacy

High frequencies (2kHz-20kHz) influence pattern complexity and particle count. Crisp highs generate intricate fractal details and increase particle density, creating delicate lace-like structures.

Rhythm: Pulsing Nodal Points

Rhythmic elements in the audio create pulsing nodal points that appear and disappear in sync with the beat. Onset detection triggers color explosions, while tempo controls the breathing rate of the entire visualization.

Physics Simulation Layers: Digital Reality

Fluid Dynamics

Colors flow with realistic viscosity and turbulence using Navier-Stokes equations. Users can adjust viscosity from water-like behavior to thick oil consistency.

Particle Systems

Touch releases cascades of luminous particles with customizable physics, each acting as an independent agent with velocity, acceleration, and lifespan.

Wave Interference

Field Dynamics

Invisible magnetic and gravitational fields influence color behavior, creating orbital motions and spiral formations as particles respond to field forces.

Technical Stack: Physics simulations run on GPU using compute shaders for real-time performance with up to 1M particles at 60fps. Integration with Matter.js for complex rigid body dynamics.

Preservation Engine
