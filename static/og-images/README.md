# Open Graph Images

This directory contains Open Graph (OG) images for social media sharing. These images appear when your website is shared on platforms like Facebook, Twitter, LinkedIn, etc.

## Required Images

All images should be **1200 x 630 pixels** (Facebook/Twitter recommended size).

### Image Files Needed:

1. **og-default.png** - Site-wide fallback image
   - Used when no specific page image is defined
   - Should represent your personal brand
   - Suggested content: Your name, title (Data Scientist & Developer), and subtle background

2. **og-home.png** - Homepage image
   - Welcome/introduction theme
   - Could feature: "Dennis J. Carroll - Data Science, Development, Creative Writing"

3. **og-projects.png** - Development Projects page
   - Technology/coding theme
   - Could feature: Code snippets, tech stack icons, project showcase

4. **og-stories.png** - Stories page
   - Creative writing theme
   - Could feature: Book/writing imagery, creative elements

5. **og-about.png** - About page
   - Personal/professional theme
   - Could feature: Your photo (optional), skills, expertise areas

## Design Guidelines

- **Dimensions:** 1200 x 630 pixels (1.91:1 aspect ratio)
- **File format:** PNG or JPG
- **File size:** Under 1MB (ideally under 300KB)
- **Safe zone:** Keep important text/imagery within center 1200 x 540px area
- **Text:** Use large, readable fonts (minimum 60px)
- **Colors:** Match your website theme (teal/dark/modern aesthetic)

## Tools for Creation

- **Canva:** Free templates for OG images
- **Figma:** Professional design tool
- **Adobe Photoshop/Illustrator:** Advanced design
- **Online generators:** Sites like og-image.xyz
