# Modern Futuristic Personal Website Design

## Overview
This project is a personal website for Dennis J. Carroll, featuring a modern, futuristic design with teal (`#008080`) as the primary color, off-white (`#F8F8F8`) as secondary, and gray accents. The design includes dynamic elements, CSS animations, and a fully responsive layout.
