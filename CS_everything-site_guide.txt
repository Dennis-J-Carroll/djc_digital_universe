## ğŸ”§ Fixing Broken Navigation (Featured Work Cards)

1. **Locate the component**

   * Search under `src/components/` or `src/sections/` for the file rendering your "Featured Work" cards.

   ```bash
   grep -R "Featured Work" -n src
   ```

2. **Swap HTML `<a>` for Gatsby `<Link>`**

   * At the top:

     ```js
     import { Link } from 'gatsby'
     ```
   * Replace:

     ```jsx
     <a href={work.link} className="..."></a>
     ```

     with:

     ```jsx
     <Link to={work.link} className="...">
       {/* child content */}
     </Link>
     ```
   * **Ensure** `work.link` matches a valid page route (e.g. `/data-science/` â†’ `src/pages/data-science.js`).

3. **Restart dev server**
   Gatsby hotâ€‘reload should pick it up, but if it doesnâ€™t, run:

   ```bash
   ```

gatsby clean && npm start

